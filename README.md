# 🌤️ Weather App — PHP CLI

A simple PHP command-line app that fetches real-time weather data using the [OpenWeatherMap API](https://openweathermap.org/api) and Guzzle HTTP client.

---

## Requirements

- PHP 8.0+
- Composer
- OpenWeatherMap API Key → [Daftar gratis di sini](https://openweathermap.org/api)

---

## Project Structure

```
Latihan Pweb/
├── src/
│   └── Weather.php       # Weather class (Guzzle + API logic)
├── vendor/               # Auto-generated by Composer (jangan di-commit)
├── Call.php              # Entry point / CLI runner
├── composer.json
├── .env                  # API key kamu (jangan di-commit)
├── example.env           # Template untuk kolaborator
└── .gitignore
```

---

## Installation

### 1. Clone repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```bash
composer install
```

### 3. Setup file environment

```bash
cp example.env .env
```

Buka `.env` dan isi dengan API key kamu:

```env
WEATHER_API_KEY=your_api_key_here
```

---

## Usage

Jalankan dari **root project** dengan nama kota sebagai argumen:

```bash
php Call.php <nama_kota>
```

### Contoh

```bash
php Call.php Surabaya
```

**Output:**

```
Mendapatkan data cuaca untuk Surabaya ...
Data cuaca untuk Surabaya:
Kota       : Surabaya
Suhu       : 31.5°C
Deskripsi  : few clouds
Kelembapan : 74%
```

Jika dijalankan tanpa argumen:

```bash
php Call.php
```

```
Penggunaan Yang Benar : php Call.php <nama_kota>
```

---

## How It Works

```
Call.php  →  baca $argv[1]  →  buat objek Weather  →  panggil getWeather($city)
                                                              ↓
                                               Guzzle GET ke OpenWeatherMap API
                                                              ↓
                                               Decode JSON → print ke terminal
```

---

## Security

> ⚠️ Jangan pernah commit file `.env` atau API key kamu ke repository.

- `.gitignore` sudah mengecualikan `vendor/` dan `.env` secara otomatis
- Gunakan `example.env` sebagai template — tidak mengandung kredensial asli
